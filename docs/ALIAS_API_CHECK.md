# エイリアス追加の楽天API確認手順

## 目的
「チ。」と「響」の楽天APIでの正確なタイトル表記を確認する

## 方法1: サイトで確認（最も簡単で確実）

### 手順
1. ブラウザでサイトを開く
2. F12キー → 開発者ツールを開く
3. `Network`タブを選択
4. 「地球の運動」で検索
5. `/api/search?q=...` のレスポンスを確認
6. JSON内の `books[0].title` の値をコピー

### 確認すべき値
以下の検索で確認：
- 「地球の運動」または「チ」→ 正確なタイトル表記
- 「響」または「小説家になる方法」→ 正確なタイトル表記

## 確認後の作業

確認したタイトルをもとに、以下のエイリアスを追加：

```typescript
// 「チ。」の場合
"チ": "[確認したタイトル]",
"ち": "[確認したタイトル]",
"チー": "[確認したタイトル]",
"地球の運動": "[確認したタイトル]",
"ちきゅうのうんどう": "[確認したタイトル]",

// 「響」の場合  
"響": "[確認したタイトル]",
"ひびき": "[確認したタイトル]",
"小説家になる方法": "[確認したタイトル]",
"しょうせつかになるほうほう": "[確認したタイトル]",
```

## 次のステップ
確認が完了したらお知らせください。エイリアスを追加します。
