追加後、私に「マージして」と言っていただければ、自動で既存のコードに統合します。

## フォーマット

以下のいずれかの形式で記載してください：

### 形式1: TypeScriptオブジェクト形式（推奨）
```typescript
{
  '略称1': '正式タイトル',
  '略称2': '正式タイトル',
}
```

### 形式2: 矢印形式
```
略称 → 正式タイトル
略称 → 正式タイトル
```

### 形式3: カンマ区切り
```
略称, 正式タイトル
略称, 正式タイトル
```
